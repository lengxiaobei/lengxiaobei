# 冷小北 Trae IDE 集成

## 概述

本插件实现了冷小北 AI 助手与 Trae IDE 的集成，通过 MCP (Model Context Protocol) 协议提供代码分析、代码生成、架构优化和自主进化等功能。

## 功能特性

- **代码分析**：分析代码结构、潜在问题和性能瓶颈
- **代码生成**：根据提示生成高质量代码
- **架构优化**：分析项目架构并提供优化建议
- **自主进化**：自动执行代码改进和架构优化

## 安装方法

1. 将本目录复制到 Trae IDE 的插件目录
2. 在 Trae IDE 中配置 MCP 服务器

## MCP 服务器配置

1. 打开 Trae IDE
2. 进入设置 → MCP 服务器
3. 点击 "添加服务器"
4. 选择 "从文件导入"
5. 选择本目录中的 `mcp-config.json` 文件
6. 保存配置

## 使用方法

### 1. 代码分析

1. 在 Trae IDE 中打开一个 Python 文件
2. 右键点击文件，选择 "Leng Xiaobei: 分析代码"
3. 查看分析结果

### 2. 代码生成

1. 在 Trae IDE 中右键点击项目目录
2. 选择 "Leng Xiaobei: 生成代码"
3. 输入代码生成提示
4. 查看生成的代码

### 3. 架构优化

1. 在 Trae IDE 中右键点击项目目录
2. 选择 "Leng Xiaobei: 优化架构"
3. 查看架构优化建议

### 4. 自主进化

1. 在 Trae IDE 中右键点击项目目录
2. 选择 "Leng Xiaobei: 运行进化"
3. 查看进化结果和变更

## 依赖项

- Python 3.14+
- 冷小北核心系统

## 配置选项

MCP 服务器配置文件 `mcp-config.json` 包含以下选项：

- `name`：服务器名称
- `description`：服务器描述
- `type`：连接类型（stdio）
- `command`：启动命令
- `cwd`：工作目录
- `env`：环境变量
- `methods`：支持的方法列表

## 故障排除

### 服务器启动失败

- 检查 Python 版本是否为 3.14+
- 检查冷小北核心系统是否正确安装
- 检查工作目录配置是否正确

### 功能执行失败

- 检查网络连接
- 检查项目路径是否正确
- 查看 Trae IDE 日志获取详细错误信息
